Movie API Documentation

This table identifies the tools (functions) available for users to utilize in the Movie API.
Included are the URLs, HTTP methods, Query Parameters, Request formats, and REsponse formats.

Business Logic URL HTTP Method Query Parameters Request Body Data Format Response Body Data Format
Return a list of ALL movies to the user /movies GET None None A JSON object containing a list of ALL movies
Returns data about a single movie by title to the user /movies/:Title GET :Title None A JSON object containing data (description, genre, director, image URL) about a single movie.
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/genre/:genreName GET :GenreName None A JSON object containing data about a specific genre.
Return data about a director (bio, birth year, death year) by name /movies/directors/:directorName GET :DirectorName None A JSON object containing data about a specific director
Return a list of ALL users to the Admin /users GET None None A JSON object containing a list of ALL movies
Allow new users to register /users POST None A JSON object containing data to add for the new user, must have the following structure:
ID: Integer,
Username: String,
Password: String,
Email: String,
Birthday: Date

A JSON object containing data about the new user.
Returns data about a single user by Username to the Admin /users GET :Username None A JSON object containing data (description, genre, director, image URL) about a single movie.
Allow users to update their user info /users/:Username PUT :Username None A JSON object containing data to update for the existing user, must have the following structure:

Username: String, (required) Password: String, (required) Email: String, (required) Birthday: Date
Allow users to add a movie to their list of favorites /users/:Username/movies/:Movietitle POST :Username/:MovieTitle None A text message indicating that the movie has been successfully added to the users favorite movies list.
Allow users to remove a movie from their list of favorites /users/:Username/movies/:MovieID DELETE :Username/:movieTitle None A text message indicating that the movie has been successfully removed from the favorite movies list.
Allow existing users to deregister /users/:id DELETE :Username None A text message indicating that the user has been successfully deregistered
